Troubleshooting Tips

  • Chrome: Look for location icon in address bar, click and select "Always allow"
  • Safari: Safari → Preferences → Websites → Location → Allow for this site
  • Firefox: Click shield icon in address bar → Allow location access
  • System: Make sure location services are enabled in your OS settings
  • HTTPS: Some browsers require HTTPS for GPS (current: )
